# Open Source Roadmap

AIMoCap uses a staged public release model.

## Available Now

- Public project page
- Public README and workflow docs
- API examples
- Output format notes
- Demo links

## Planned

- More client examples
- Small helper scripts for result inspection
- More sample output files
- Community integration notes for animation and robot workflows

## Not Planned for This Repository

- Hosted service source
- Production processing stack
- Account and billing systems
- Private model assets
- Deployment configuration
